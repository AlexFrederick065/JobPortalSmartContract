Overview
This Smart Contract is build on Ethereum Blockchain. It has the fucntionality from adding a job and allowing participant to apply these jobs.

Structure
- Constructor
Contract has a constructor to store the address form which the contract is being deployed.
- Structs
The contract has two structs such as applicant_details for applicants details, job_details for job details.
- Mapping
Contract has 3 mappings for efficient retrieval and manipulation of data.
- Modifier
Contract has 3 Modifier for checking few conditions
- Enum
Contract has 1 enum that is used to specific jobType

Function
- Add Applicant
- Fetch Applicant
- Fetch Applicant Type
- Add Job
- Fetch Job
- Apply to a Job
- Add Rating of an applicant
- Fetch Rating of an applicant
